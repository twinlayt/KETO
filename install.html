Keto Landing Page - Kurulum Rehberi

Gereksinimler
  • PHP 7.4+ (Önerilen: PHP 8.0+)
  • MySQL 5.7+ veya MariaDB 10.2+
  • Apache/Nginx web sunucusu
  • mod_rewrite aktif olmalı

1. Database Kurulumu

cPanel/phpMyAdmin'de:

  1. Yeni database oluşturun: keto_landing
  2. MySQL kullanıcısı oluşturun ve database'e tam yetki verin
  3. database/setup.sql dosyasını import edin
Manuel SQL: phpMyAdmin'de SQL sekmesine gidin ve database/setup.sql içeriğini yapıştırın.

2. Yapılandırma

api/config.php dosyasını düzenleyin:

define('DB_HOST', 'localhost');           // MySQL host
define('DB_NAME', 'keto_landing');        // Database adı
define('DB_USER', 'your_mysql_username'); // MySQL kullanıcı
define('DB_PASS', 'your_mysql_password'); // MySQL şifre
define('SITE_URL', 'https://yourdomain.com'); // Domain

3. Dosya Yükleme

Sunucunuza yükleyin:

  • Tüm dosyaları public_html/ veya www/ klasörüne
  • uploads/ klasörüne yazma izni verin (chmod 755)
  • .htaccess dosyasının yüklendiğinden emin olun

4. Test

Bu URL'leri test edin:

  • https://yourdomain.com/ - Ana sayfa
  • https://yourdomain.com/api/test - Database test
  • https://yourdomain.com/panel.html - Admin panel

5. Admin Erişimi

  • URL: https://yourdomain.com/panel.html
  • Kullanıcı: admin
  • Şifre: admin123
Güvenlik: İlk girişten sonra şifreyi değiştirin!

Yaygın Sorunlar

Çözüm: cPanel'de PHP sürümünü 7.4+ yapın
PHP 8.0 veya PHP 8.1 önerilir

Kontrol edin:
  • MySQL kullanıcı adı ve şifre doğru mu?
  • Database adı doğru mu?
  • MySQL kullanıcısının database'e erişim yetkisi var mı?
  • Host adresi doğru mu? (localhost, 127.0.0.1 veya sunucu IP)

Terminal/SSH ile: 
chmod 755 uploads/
chmod 644 api/*.php
chmod 644 *.html
Kurulum Tamamlandı!

Artık siteniz canlı! Admin panelinden içerikleri yönetebilirsiniz.
